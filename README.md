# KeaShell - A general purpose utility shell in NodeJS

## List of currently supported commands:
1. ls
2. cd
3. pwd
4. help
5. history
6. clear
7. exit
8. touch
9. mkdir
10. rm
11. cp
12. cat
13. mv
14. grep
15. wc

## Installation
```
 npm run build
 (Change the targets in the package.json build script if any other OS)
```

How to use,
```
 ./index.exe or ./index
```
